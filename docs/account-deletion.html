계정·데이터 삭제 안내

앱: 그날 우리 (com.ourday.our_day) · 최종 수정일: 2026년 7월 1일

‘그날 우리’는 별도의 회원가입이나 계정이 없습니다. 모든 사진과 기록은 사용자의 기기에 저장되며, 당사 서버에는 어떤 개인정보도 보관하지 않습니다. 따라서 데이터 삭제도 모두 사용자가 직접, 즉시 할 수 있습니다.

1. 기기에 저장된 데이터 삭제

  1. 앱 안에서 개별 사진·기록을 삭제할 수 있습니다.
  2. 모든 데이터를 한 번에 지우려면 앱을 삭제(제거)하면 됩니다. 앱과 함께 저장된 모든 사진·기록이 기기에서 제거됩니다.
  3. 또는 기기 설정 → 앱 → ‘그날 우리’ → 저장공간 → 데이터 삭제를 선택해도 됩니다.

2. 구글 드라이브 백업을 사용한 경우

클라우드 백업을 켰던 분은, 백업 파일이 본인 구글 드라이브의 앱 전용 숨김 폴더에 저장돼 있습니다. 다음 방법으로 완전히 삭제할 수 있습니다.

  1. 앱 설정 → 클라우드 백업 → 연결 해제 및 백업 삭제.
  2. 또는 구글 계정에서 앱 접근 권한을 직접 회수: myaccount.google.com/permissions → ‘그날 우리’ → 액세스 권한 삭제. 권한을 회수하면 앱이 만든 숨김 폴더 데이터도 함께 정리됩니다.

3. 삭제되는 데이터 / 보관되는 데이터

구분내용보관 기간
삭제되는 데이터사진, 메모, 키·나이 기록, 위치(회상 알림용), 설정, 구글 드라이브 백업 파일위 절차 즉시 삭제
당사 서버 보관 데이터없음 (수집·저장하지 않음)해당 없음
