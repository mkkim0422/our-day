## 13. 콘텐츠 등급 설문(IARC) — 권장 답안

대부분 **"아니요"** 입니다. 이 앱은 사진 기록 도구이며 폭력·성·도박·욕설 콘텐츠가 없습니다.

| 질문 범주 | 답 |
|---|---|
| 폭력 / 사실적 폭력 | 아니요 |
| 성적 콘텐츠 / 노출 | 아니요 |
| 욕설·비속어 | 아니요 |
| 약물·알코올·담배 묘사 | 아니요 |
| 도박(실제/모의) | 아니요 |
| 공포·무서운 콘텐츠 | 아니요 |
| 사용자 간 소통(채팅/메시지) | 아니요 (앱 내 사용자 간 소통 없음) |
| 사용자 생성 콘텐츠 공유 | 아니요 (외부 공유는 OS 공유시트로 사용자가 직접, 앱 내 게시 없음) |
| 위치 공유 | 아니요 (위치는 기기 내 회상 알림용, 외부 공유 없음) |
| 개인정보 수집·공유 | 데이터 보안 양식과 동일하게 응답 |
| 디지털 구매(인앱결제) | 아니요 (v1 IAP 없음) |

→ 예상 등급: **전체 이용가(3+)**.

## 15. 권한 선언 (민감 권한) — 콘솔 "앱 콘텐츠" 양식

### (가) 위치 권한
- Play 콘솔에서 위치 권한 사용 시 **권한 선언서** 작성 필요.
- 핵심 문구(붙여넣기용):
  > "위치는 사용자가 선택적으로 켜는 '장소 기반 회상 알림' 기능에만 사용됩니다.
  > 특정 장소 근처에서 그곳의 과거 사진을 떠올려 주는 용도이며, 좌표는 기기에만
  > 저장되고 외부로 전송하거나 광고·추적에 사용하지 않습니다. 백그라운드 위치는
  > 사용하지 않습니다(전경/사용 중에만)."
- 백그라운드 위치(ACCESS_BACKGROUND_LOCATION) **사용 안 함** → 해당 선언 불필요.

### (나) 사진·미디어 권한 (READ_MEDIA_IMAGES)
- 용도: 갤러리 사진 백필(과거 사진 정리)·스토리 자동 생성.
- 핵심 문구:
  > "기존 갤러리 사진을 골라 날짜에 맞춰 정리(백필)하고, 같은 구도 기록을
  > 자동으로 묶어 보여주기 위해 사진 접근 권한을 사용합니다. 선택한 사진만
  > 앱 내에서 처리하며 외부로 전송하지 않습니다."

### (다) 카메라 / 알림
- 표준 권한(별도 선언 보통 불필요). 카메라=촬영, 알림=촬영 리마인더·회상 알림.

### 광고 ID(AD_ID)
- 광고 SDK 없음 → **com.google.android.gms.permission.AD_ID 미사용**. 콘솔에서 "광고 ID 사용 안 함" 선언.

## 16. 앱 액세스(리뷰어용 안내) — 콘솔 "앱 액세스" 양식

- 앱의 **모든 기능은 로그인 없이** 사용 가능 → 콘솔에서 **"모든 기능을 특별한 접근 없이 사용 가능"** 선택 가능.
- 단, **클라우드 백업** 기능만 사용자 본인 구글 계정 로그인이 필요. 리뷰어 안내문(붙여넣기용):
  > "이 앱은 회원가입/로그인 없이 모든 핵심 기능(촬영·오버레이·타임랩스·비교·꾸미기·백필)을
  > 사용할 수 있습니다. '클라우드 백업'만 사용자 본인 구글 드라이브 로그인이 필요하며,
  > 이는 선택 기능입니다. 별도 테스트 계정은 필요하지 않습니다."
- (참고) OAuth가 테스트 모드면 리뷰어 구글 계정이 테스트 사용자에 없어 백업 로그인은 막힐 수 있음.
  공개 심사 전 OAuth 동의화면 게시 또는 리뷰어 계정 추가 고려(체크리스트 17·18).
